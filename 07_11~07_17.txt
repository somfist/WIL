1. JS 함수 익숙해지기
	a. Set : 활용방안중에 하나로 배열을 Set형태로 변경시 유니크한 값만 남음
	b. reduce(acc,v) : acc에 v의 값을 더해 반환해줌 => 반환값은 객체임
	c. Math.pow : 제곱근
	d. Math.max/min : 기본적으로 나열한 데이터 중 최대/최소 값을 찾아주는데 배열을 넣으면 나열한 데이터가 아니기에 전개연산자(스프레드)와 같이쓰거나 .apply로 쓴다
		d-1. .apply는 아직 이해가 부족해 좀더 찾아봐야될듯 하다
	e.charCodeAt / fromCharCode : 아스키코드로 변환하고 다시 문자로 변환할때
	f. filter : forEach와 비슷하게 작동하지만 조건에 따라 []을 반환함, 속성값으로 엘리먼츠, 인덱스, 소스등이 있어 활용가능

2. JS 변수 익숙해지기
	a. let, const : 재할당 가능, 불가인데 저장할 때 함수로도 저장가능
